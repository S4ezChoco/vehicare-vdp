§0 Project Overview

VDP OBD-II is a multi-platform vehicle diagnostics and environmental monitoring system. It integrates Arduino-based environmental sensors, OBD-II ELM327 vehicle diagnostics, a local AI diagnostic engine, and a real-time web dashboard into a unified platform.
